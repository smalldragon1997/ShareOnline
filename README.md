# shareOnline
在线账号分享平台

注意：该项目部分功能并未完成

使用技术栈：
  1.SSM框架作为项目的后端环境，提供数据接口，使用Jsp作为访问页面
  2.使用ajax获取后端数据以及jquery+bootstrapUI界面的样式渲染
